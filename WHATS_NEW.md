# What's New in QuickServe

## [1.3.0] (2026-03-31)

### Added

- **HMR for Markdown** — Markdown files now support Hot Module Replacement, so edits reflect instantly in the browser without a full page reload
- **Improved Markdown preview** — preview now renders in a clean GitHub-style layout with minimal styling and full dark mode support

### Fixed

- **node_modules excluded from file explorer** — Excluded `node_modules` from the directory parsing engine, reducing overhead when working in larger projects
